
## Usage

An example `token.js` file could be:
```js
module.exports ={
    // past your telegram bot token like
    token: '4964312452:NeN554GslMAAcyM2XrgcSW53nWIUgHG0-zw'
}
```
then run

```sh

node index.js

```
